# SMEH

SMEH (Satisfactory Modding Environment Helper) is a small console app that helps you set up the Satisfactory modding environment on Windows. It installs : Visual Studio 2022, the Clang toolchain, the CSS Unreal Engine, the SML starter project, and Wwise.

Build with `dotnet build` and run the exe, or use `dotnet run`. 

Temp files end up under `%TEMP%\SMEH`.
